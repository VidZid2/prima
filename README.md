# PRIMA Digital Technology Solutions

PRIMA is a high-performance, premium web platform engineered for a leading digital technology solutions agency. Designed to make a powerful first impression, the site showcases a sophisticated dark mode aesthetic, advanced micro-animations, and custom interactive features built specifically to display corporate services, seminars, training, and modern integrations.

---

## 🎨 Visual & Technical Excellence

* **Premium Dark Mode Theme:** A meticulously curated black-and-cyan palette tailored for maximum visual depth, featuring inset border highlights and custom depth card effects.
* **Fluid Smooth Scrolling:** Integrated with the **Lenis** smooth-scrolling engine for modern, high-friction, fluid layouts.
* **Immersive Animation System:** Powered by **Framer Motion** for sleek page transitions, element-in-view reveals, and 3D tactile button responses.
* **Interactive Modals:** A step-by-step custom booking and inquiry funnel modal that securely handles user inputs and locks page scrolling on both desktop and mobile devices.
* **Bespoke Design Components:** Custom date pickers, tactile badges with 3D gradient borders, and custom typography utilising `@clear-sans` and `@mokoto` fonts.

---

## 💻 Tech Stack

* **Core Framework:** Next.js (App Router, React 19)
* **Styling Engine:** Tailwind CSS & Vanilla CSS
* **Animations:** Framer Motion (Motion for React)
* **Scroll Engine:** Lenis / ReactLenis
* **Icons & Assets:** Lucide React
